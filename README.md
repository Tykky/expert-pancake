scripts:

## How to build & run
Works like any other node project. Install
dependencies with:
```
npm install
```
run in production mode with:
```
npm start
```
The backend can be run in dev mode with:
```
npm run dev-backend
```
and frontend in dev mode with:
```
npm run dev-frontend
```
